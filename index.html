Projects

Since 2015 I have worked on various projects some of which have made to the Apple AppStore, some have been made private due to the nature of the contracts and some I have uploaded to my GitHub.

Please find some examples below:
AVERAGIZER: AVERAGE CALCULATOR
Average calculating app. It uses boards to calculate the average product made per hour. The user is required to input a product name onto a board. Next add the times from and times to he/she was making a product. If it happens that more than one product was made in a 24h period, the user can add more boards to be able to calculate the averages per product.
The app has been built using a complete programmatic approach without StoryBoards, UIKit, Custom made scrollable and resizable views.


PIECEPAY: PIECEWORK CALCULATOR
PiecePay is a piecework calculator and tracker app for people who are being paid per piece, it also allows to set a minimum hourly rate when the user is not being paid on piecework scheme. It calculates the day rate earned, the rate per hour achieved as well as shows how much more you have earned on that day then your normal average daily minimum rate. The user has the option to save the calculations into a Realm database and is able to edit the calculations if needed. The saved calculations also show a monthly summary of earnings and products produced per month.
The app has been made completely programmatically using UIKit without StoryBoards, Persistent Storage as a Realm Database, Custom UI Elements, Unit tests, UI Tests, CocoaPods


SHAPE SWAP: MATCH 4 PUZZLE
Shape Swap is a puzzle game similar to those match 3 games, however in this game the player is required to match 4 instead of 3 and gets penalty points for breaking this rule. To make the game easier once you move a shape it stays in that place unlike the other match 3 games where the shape returns to it's original position if there is no match.
Game created using: SpriteKit with implemented Game Center and Google Ads.


SIMON SAYS: COLOURBLIND EDITION
Classical Simon Says game where the player is required to repeat the sequence that the game is producing. It gets harder and harder with every iteration. The player can play against their friend by sharing the phone. There is an option to login to Game Center and this way the player's high score gets submitted to the high score board, which gets populated and keeps the best 25 scores.
Game created using: UIKit with implemented Game Center, Google Ads, custom HighScore board populated over the internet.


FUEL CALCULATOR: MPG CONVERTER
Born rather out of frustration, I am not used to think in terms of fuel economy in MPGs instead I prefer to use L/100km. I have seen many apps but they lacked simplicity or have missed some basic functionality. This app is both simple and has 3D touch shortcuts for convenience. The user just needs to enter the conversion value and tap convert. There is also an option to track fuel economy by saving everything into a Realm database.
Fuel conversion and tracking app: UIKit, Persistent Storage using Realm Database, 3D Touch.


SANTA'S: PRESENT MIXUP
It is a similar game to Shape Swap where the player has to match 4 shapes instead of 3 in order to proceed to the next level, however this has a Christmas theme. Game created using: SpriteKit with implemented Game Center, Google Ads and Facebook login integration.


CLIMA: WEATHER APP
Weather app, which allows you to check the weather based on your GPS location or of a chosen city. Built using: UIKit, URLSession, CoreLocation, RESTful Api Integration from OpenWeather. JSON Decoding.
Source Code on GitHub - Clima-Pure
Also available in Objective-C at Clima-ObjC
